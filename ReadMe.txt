Программа для занесения гостей в базу данных ParsecNet3 и генерация для них QR
Данные заносятся в текстовый файл (формат файла .txt)
Пример того как должны выглядеть данные "Комарова Лидия Филиповна;28.07.2024;28.08.2024;Посещение конфиренции;"
Для запуска программы нажмите на файл Script.exe по пути"C:\Users\valer\Downloads\Script_QR\Script\Script\bin\Release\net8.0\Script.exe"
При работе с программой важно знать:
- Место расположения файла с данными
- Данные необходимые для авторизации или использовать авторизацию по умолчанию
- Название подразделения куда будут занесены гости
- Название Группы доступа (в которой находятся контроллеры через которые гости смогут проходить)
Файл (QR_photo), где хранятся картинки QR доступны по пути "C:\...\Script_QR\QR_photo"
Во время выполнения программы все действия записываются в Лог файл (Logfile), он доступен по пути "C:\...\Script_QR\Logfile.txt"
Так же в лог файле доступны тексты для генерации QR кодов для каждого гостя
